# Data Analytics & Visualization Portfolio

A modern, responsive portfolio website showcasing data analytics and visualization expertise. Built with pure HTML5 and CSS3 for fast load times and smooth interactions.

## Features

✨ **Modern Design**
- Dark theme with vibrant accent colors (teal & blue)
- Smooth scroll animations and fade-up effects
- Glass-morphism UI elements with backdrop blur
- Fully responsive mobile design

📱 **Responsive Layout**
- Desktop, tablet, and mobile optimized
- Fluid typography and grid systems
- Touch-friendly navigation

🎯 **Sections**
- **Hero** — Compelling introduction with featured stats
- **Skills** — Core capabilities with technology tags
- **Portfolio** — Showcase of featured projects
- **About** — Personal background and expertise highlights
- **Contact** — Multiple ways to get in touch
- **Footer** — Site information

⚡ **Performance**
- Single-page application (no external dependencies)
- Inline CSS for minimal HTTP requests
- Intersection Observer API for efficient animations
- Optimized for Core Web Vitals

## Usage

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize content (name, links, projects, etc.)

## Customization

### Update Contact Information
Replace placeholder links in the contact section:
- `mailto:samuel@example.com` → Your email
- LinkedIn, GitHub, and CV links

### Modify Content
Edit these sections in `index.html`:
- Hero headline and description
- Skills and tools
- Portfolio projects
- About text
- Social links

### Adjust Colors
Modify CSS variables at the top of the `<style>` block:
```css
:root {
  --accent: #00e5a0;      /* Primary accent color */
  --accent2: #0ea5e9;     /* Secondary accent color */
  --bg: #0a0c10;          /* Background color */
  /* ... more variables */
}
```
